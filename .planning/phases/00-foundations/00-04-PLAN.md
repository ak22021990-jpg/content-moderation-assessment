<!-- Cloudflare R2 CDN decision (D-15 CORRECTION from 00-RESEARCH.md) -->

CRITICAL: jsDelivr proxies raw GitHub URLs. For LFS files, GitHub serves the 128-byte pointer file at the raw URL — not the binary. jsDelivr therefore serves the pointer, not the video. This is a confirmed open issue since 2020 (github.com/jsdelivr/jsdelivr/issues/18235).

Production CDN for video MUST be Cloudflare R2 (or direct LFS via GitHub Pages for low-volume use).
R2 free tier: 10 GB storage, 1M Class A writes/mo, 10M Class B reads/mo, $0 egress.

<!-- Encoding spec for real clips (CONTENT-04) — document in video-manifest.md -->

Target encoding for all real V1-V5 clips (Phase 2 onward):
  H.264, 720p (1280×720), CRF 26, AAC 96 kbps, -movflags +faststart, ≤ 20 MB per clip
  ffmpeg command template:
    ffmpeg -i input.mp4 -c:v libx264 -crf 26 -preset medium -vf scale=1280:720 -c:a aac -b:a 96k -movflags +faststart output.mp4
