scripts/generate-sprites.mjs contract:
  - Arg-free: reads playlist.json from ../src/data/playlist.json (relative to script)
  - For each video in playlist.json:
    1. ffprobe -v error -show_entries format=duration -of csv=p=0 "public/videos/{id}.mp4" → duration float
    2. Compute COLS=10, ROWS=10, TILE_W=160, TILE_H=90, interval = duration / (COLS * ROWS)
    3. ffmpeg -y -i "public/videos/{id}.mp4" -vf "fps=1/{interval},scale={TILE_W}:{TILE_H},tile={COLS}x{ROWS}" -frames:v 1 "public/sprites/{id}_sprite.jpg"
    4. Generate thumbs.vtt: WebVTT header + COLS*ROWS cues mapping frame seconds → sprite#xywh=tileX,tileY,TILE_W,TILE_H
    5. Generate chapters.vtt: WebVTT header + per-chapter zero-duration cues from playlist.json chapters[]
    6. Write to public/vtt/{id}.thumbs.vtt and public/vtt/{id}.chapters.vtt
  - Output: stdout progress lines; non-zero exit on error
  - Requirements: ffmpeg + ffprobe on PATH (CI installs via apt-get; dev verifies locally)

WebVTT thumb format (thumbnails):

  WEBVTT

  00:00:00.000 --> 00:00:03.600

  00:00:03.600 --> 00:00:07.200
  /sprites/v01_sprite.jpg#xywh=160,0,160,90
