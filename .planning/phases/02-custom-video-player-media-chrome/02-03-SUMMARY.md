# Summary — Wave 3 (Advanced Player)

Sprite thumbnail generation, chapter markers, custom keyboard controls, and CI integration.

## What Was Built

### Task 02-03-01: ffmpeg Sprite Generation Script
- `scripts/generate-sprites.mjs` — Node.js ESM script
  - Exports pure functions: `formatTime`, `generateThumbsVtt`, `generateChaptersVtt`
  - CLI mode: reads `playlist.json` → `ffprobe` for duration → `ffmpeg` sprite generation → writes VTT files
  - `fps=1/{interval},scale=160:90,tile=10x10` filter pattern per RESEARCH.md
- `tests/scripts/generate-sprites.test.js` — 18 pure function unit tests
  - formatTime edge cases (0, 5.5, 65, 3661.123, 3599.999)
  - generateThumbsVtt: 100 cues, correct xywh coordinates, monotonic ordering
  - generateChaptersVtt: zero-duration cues, label text, empty array handling
  - Brand safety: zero forbidden strings in generated VTT

### Task 02-03-02: CI Deploy Workflow
- `.github/workflows/deploy.yml` — 4 new steps between `npm ci` and `npm run build`:
  1. `sudo apt-get install -y ffmpeg`
  2. `node scripts/generate-sprites.mjs`
  3. Verify generated assets exist (test -f)
  4. Verify sprite is real JPEG (not LFS pointer)

### Task 02-03-03: Track Elements + Keyboard Controls
- `<track default label="thumbnails" kind="metadata">` — VTT thumbnail previews
- `<track default kind="chapters" srcLang="en">` — chapter markers on seek bar
- Custom keyboard handler:
  - Disables built-in ArrowLeft/ArrowRight (10s default) via `controller.hotkeys.add('noarrowleft', 'noarrowright')`
  - ArrowLeft: seek -5s (clamped at 0)
  - ArrowRight: seek +5s (clamped at duration)
  - `e.preventDefault()` on arrow keys to prevent page scroll
  - Space / M / ArrowUp / ArrowDown pass through to media-chrome defaults
  - Cleanup removes listener on unmount

### Task 02-03-04: Test Video Assets + LFS
- `public/videos/v01.mp4` — test video (LFS-tracked)
- `public/sprites/v01_sprite.jpg` — 10x10 thumbnail sprite grid (LFS-tracked, verified JPEG via FF-D8 magic bytes)
- `public/vtt/v01.thumbs.vtt` — WebVTT metadata track with 100 xywh cues
- `public/vtt/v01.chapters.vtt` — WebVTT chapters track with 3 zero-duration cues from `playlist.json`
